# System Overview

## Purpose

This starter repository models the Harness Engineering process itself. It can
be copied into a product repository and customized with real domain knowledge.

## Current System

- Runtime: Node.js 20 or newer.
- External dependencies: none.
- Validation entry point: `npm run validate`.
- Test entry point: `npm test`.

## Adoption Guidance

Start with one real product flow and document only the rules that affect that
flow. Add wiki pages when a requirement needs business context that is not
already visible in code or tests.
